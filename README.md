# 🚀 SmartVenue Antigravity System

A smart, lightweight, AI-inspired venue assistant that improves crowd movement, reduces waiting time, and enhances user experience inside large stadiums and events.



## 📌 Problem Statement

Large events like stadiums, concerts, and festivals often suffer from:
- Heavy crowd congestion at entry/exit gates
- Long waiting times for food and services
- Confusion in navigation inside the venue

Users waste time and experience frustration due to lack of real-time guidance.



## 💡 Solution

SmartVenue Antigravity is a **web-based intelligent assistant prototype** that simulates smart decision-making for venue management.

It helps users:
- Choose less crowded entry gates
- Estimate food queue waiting time
- Find the best exit route based on user input

The system uses logic-based “smart simulation” to mimic real-world crowd prediction and guidance.



## ✨ Features

- 🧭 Smart Crowd Prediction System (Gate Recommendation)
- 🍔 Food Queue Time Estimation
- 🚪 Exit Route Suggestion based on user input
- 🎯 Context-aware decision making (A / B / C zones)
- ⚡ Fast, lightweight, and mobile-friendly UI
- 🎨 Modern glassmorphism design



## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism UI design)
- JavaScript (Logic-based AI simulation)
- GitHub Pages (Deployment)


## 🧠 How It Works

1. User enters their gate/zone (A, B, or C)
2. System calculates:
   - Crowd level score (0–100)
   - Food waiting time (dynamic estimation)
   - Best exit route based on zone
3. Outputs a smart recommendation like a digital assistant


## 📊 Example Flow

- Input: Gate A  
- Output:
  - Crowd Score: 25% → Low Crowd 🟢
  - Food Queue: 4 mins → Fast Service 🟢
  - Exit: Exit 1 (Fastest Route 🟢)

---

## 📌 Assumptions

- Venue data is simulated using logic-based conditions
- Crowd levels are generated using smart estimation (not real-time sensors)
- System is designed as a prototype for smart city/event innovation
